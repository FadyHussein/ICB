# Teacher User Guide

## ICB Sunday School Attendance Tracking System

Welcome! This guide will help you quickly and easily submit attendance for your Sunday School class.

**What you need:**
- A tablet or computer

- Web browser (Chrome, Safari, Edge, or Firefox)

- About 2-3 minutes per class

**When to use:**
- Every Sunday after class
- Submit attendance for each class you teach

## Accessing the System

### Step 1: Connect to WiFi

Make sure your device is connected to the ICB network:
- WiFi Network: **ICB-Network** (or as configured)
- Look for the WiFi icon on your device
- If not connected, select the network and enter the password

### Step 2: Open Your Browser

- **iPad/iPhone**: Tap the Safari icon
- **Android Tablet**: Tap the Chrome icon
- **Computer**: Open Chrome, Edge, or Firefox

### Step 3: Go to the Application

Type this address in the browser:
```
http://[IP-ADDRESS]:5173
```

**Your specific address:**
```

(Ask your administrator for the exact address)
```

**Tip:** Bookmark this address for easy access next time!

## Step-by-Step Instructions

### Screen 1: Select Your Program

**What you see:**
- Two program cards: **Iqra** and **Islamic Studies**

**What to do:**
1. Tap the card for the program you're teaching
2. Example: If teaching Iqra, tap "Select Program" on the Iqra card

---

### Screen 2: Select Your Level

**What you see:**
- Multiple level cards (Level 1, Level 2, etc.)
- Each card shows how many students are in that level

**What to do:**
1. Find the level you're teaching
2. Tap "Select Level" on that card

**Example:**
- Teaching Iqra Level 2? Tap the "Level 2" card

**Going Back:**
- Tap the ← back arrow to choose a different program

---

### Screen 3: Select Your Name

**What you see:**
- List of teacher names
- Search box at the top

**What to do:**

**Option 1 - Scroll and Find:**
1. Scroll through the list of teachers
2. Find your name
3. Tap "Select Teacher" next to your name

**Option 2 - Use Search:**
1. Tap in the search box
2. Type your first or last name
3. Your name will appear
4. Tap "Select Teacher"

**Example:**
- Searching for "Ahmed": Type "ahmed" in search box

---

### Screen 4: Mark Attendance

This is the main screen where you mark who was present.

**What you see:**

**Top Section:**
- Today's date
- Program name (e.g., "Iqra Program")
- Level (e.g., "Level 2")
- Your name
- Week number

**Student List:**
- All students in this class
- Each student has:
  - Name
  - Present/Absent button
  - Page number box (for present students)
  - Notes box

**Bottom Buttons:**
- "Select All" - Marks everyone present
- "Clear All" - Marks everyone absent
- "Submit Attendance" - Saves your work

---

### How to Mark Attendance

**For Each Student:**

**If Student Was PRESENT:**
1. Tap the "Absent" button (red with ✗)
2. It changes to "Present" (green with ✓)
3. If they finished pages today:
   - Tap the page number box
   - Enter the page number they completed (e.g., "15")
4. Add any notes if needed

**If Student Was ABSENT:**
- Leave as "Absent" (red button)
- No need to do anything else

**Example:**
```
Student: Ali Mohamed
- Tap "Absent" → Changes to "Present" ✓
- Tap page box → Enter "12"
- Add note: "Completed well"
```

---

### Quick Methods

**Everyone Was Present:**
1. Tap "Select All" button
2. All students marked present
3. Add page numbers for individuals as needed

**Everyone Was Absent (snow day, etc.):**
1. Tap "Clear All" button
2. All students marked absent
3. Tap "Submit Attendance"

**Mixed Attendance:**
1. Tap "Select All" first
2. Then tap specific students to mark them absent
3. This is faster than marking one-by-one

---

### Adding Notes

**When to add notes:**
- Student behavior (positive or concerning)
- Special achievements
- Important information for next week
- Medical issues or absences

**How to add notes:**
1. Tap in the notes box for the student
2. Type your note
3. Keep it brief but informative

**Examples of good notes:**
- "Memorized new Surah!"
- "Having difficulty with letter Qaf"
- "Absent due to illness"
- "Excellent participation today"

---

### Screen 5: Submit Your Attendance

**When you're done marking everyone:**

1. **Review your work:**
   - Check all students are marked correctly
   - Verify page numbers are accurate
   - Read over any notes

2. **Tap "Submit Attendance":**
   - The button turns blue and shows "Submitting..."
   - Wait a few seconds
   - Don't tap it again!

3. **Wait for confirmation:**
   - Screen changes to show success message
   - Shows summary of your submission

---

### Screen 6: Confirmation

**What you see:**
- ✓ Success message
- Summary:
  - Total students
  - How many were present
  - How many were absent
  - Date and time

**What to do:**

**If teaching another class:**
- Tap "Submit Another Attendance"
- Goes back to start
- Repeat process for next class

**If finished:**
- Tap "Go Home"
- You're done!
- Can close the browser

---

## Tips for Efficient Use

### ⚡ Speed Tips

1. **Use "Select All" for most-present days:**
   - Tap "Select All"
   - Unmark the few who were absent
   - Much faster than marking everyone individually

2. **Bookmark the application:**
   - First time: Add bookmark in browser
   - Next time: Just tap bookmark
   - Saves typing the address

3. **Add to home screen (tablets):**
   - Makes it like a regular app
   - One tap to open
   - **iPad:** Tap share icon → "Add to Home Screen"
   - **Android:** Menu → "Add to Home Screen"

4. **Pre-fill common page numbers:**
   - If most students finish same page
   - Mark all present
   - Enter page numbers quickly

### 📱 Device Tips

**Using a Tablet:**
- Hold in portrait mode for easier scrolling
- Use two hands for faster input
- Increase text size in browser if needed

**Using a Phone (if necessary):**
- Works but smaller screen
- Rotate to landscape for wider view
- Scroll carefully

**Using a Computer:**
- Can use keyboard for faster notes
- Tab key to move between fields
- Enter key to submit

### ✅ Accuracy Tips

1. **Take attendance during class:**
   - Keep a paper list if needed
   - Mark as you go
   - Enter in system after class

2. **Double-check before submitting:**
   - Count present vs. your head count
   - Verify page numbers make sense
   - Review notes for clarity

3. **Submit right after class:**
   - While memory is fresh
   - Don't wait until end of day
   - Less chance of forgetting

### 🔄 Multiple Classes

**If teaching multiple classes:**

1. Complete first class
2. Tap "Submit Another Attendance"
3. Select different program or level
4. Mark attendance
5. Submit
6. Repeat as needed

**Keep track:**
- Check off each class as you submit
- Use the confirmation screen to verify

## Common Questions (FAQ)

### Q: What if a student arrived late?

**A:** Mark them as present. The system doesn't track arrival time, only whether they attended.

---

### Q: What if I make a mistake?

**A:** 
- **Before submitting:** Just change it on screen
- **After submitting:** Contact the administrator to correct it in the Excel file

---

### Q: What if I forget to submit attendance?

**A:**
- Submit it as soon as you remember
- The date is automatic (today's date)
- Add a note explaining: "Late submission, class was on [date]"
- Contact administrator if you need the date corrected

---

### Q: Can I edit attendance after submitting?

**A:**
- Not through the web app
- Contact administrator to make changes in Excel file
- This is by design to prevent accidental changes

---

### Q: What if the system is slow or not responding?

**A:**
1. Check your WiFi connection
2. Refresh the browser page
3. Try again
4. If still not working, contact administrator

---

### Q: What if I don't see my name in the teacher list?

**A:**
- Use the search box to look for your name
- Check spelling
- If truly not there, contact administrator to be added
- They need to add you to the Excel file

---

### Q: What if a student is not in the list?

**A:**
- Verify you selected the correct program and level
- If student is genuinely missing, contact administrator
- They need to add student to the Excel file

---

### Q: Do I need to save my work?

**A:**
- No auto-save while you're working
- Data is only saved when you tap "Submit Attendance"
- Don't close browser until you submit

---

### Q: Can I go back and change after submitting?

**A:**
- No, submissions are final in the web app
- This prevents accidental overwrites
- Contact administrator for corrections

---

### Q: What's the week number for?

**A:**
- Tracks which week of the year it is
- Updated automatically by administrator
- Used for progress tracking throughout the year

---

### Q: Do I need internet access?

**A:**
- You need local network access (WiFi)
- Don't need internet connection
- System runs on local network only

---

### Q: Can I use my phone?

**A:**
- Yes, but tablet or computer is better
- Smaller screen can be harder to use
- All features work the same

## Troubleshooting

### Problem: Can't access the application

**Symptoms:**
- Browser says "Cannot connect"
- Page doesn't load

**Solutions:**
1. ✅ Check WiFi is connected to ICB network
2. ✅ Verify the URL is correct
3. ✅ Ask administrator if server is running
4. ✅ Try closing and reopening browser
5. ✅ Restart your device

---

### Problem: Student list is empty

**Symptoms:**
- Selected program and level
- No students showing

**Solutions:**
1. ✅ Verify you selected the correct level
2. ✅ Go back and reselect
3. ✅ Refresh the page
4. ✅ Contact administrator if problem persists

---

### Problem: Submission fails

**Symptoms:**
- Click "Submit Attendance"
- Error message appears
- Data not submitted

**Solutions:**
1. ✅ Check WiFi connection
2. ✅ Don't click submit button multiple times
3. ✅ Wait a minute and try again
4. ✅ If error mentions "file locked", ask if someone has Excel file open
5. ✅ Contact administrator if continues

---

### Problem: Page is slow

**Symptoms:**
- Takes long time to load
- Buttons slow to respond

**Solutions:**
1. ✅ Check WiFi signal strength (move closer to router)
2. ✅ Close other apps on your device
3. ✅ Restart your browser
4. ✅ Try a different browser
5. ✅ Notify administrator

---

### Problem: Can't enter page numbers

**Symptoms:**
- Page number box not accepting input
- Numbers don't appear

**Solutions:**
1. ✅ Make sure student is marked "Present" first
2. ✅ Tap directly in the page number box
3. ✅ Try using on-screen keyboard
4. ✅ Refresh page and try again

---

### Problem: Accidentally submitted wrong data

**Symptoms:**
- Submitted attendance
- Realized you made mistakes

**Solutions:**
1. ✅ Note what needs to be corrected
2. ✅ Contact administrator immediately
3. ✅ Provide: Date, Program, Level, Student names, what needs correcting
4. ✅ Administrator will fix in Excel file
